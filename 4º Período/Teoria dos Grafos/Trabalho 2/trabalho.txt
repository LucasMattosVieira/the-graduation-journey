Neste trabalho, você somente é permitido usar os módulos de grafos colocados na pasta 
Material de Aula>Trabalho do nosso grupo. Qualquer outra biblioteca sobre grafos não é permitida.

O trabalho consiste em implementar uma série de funções colocadas no módulo Grafo.hs, 
criado por você,  cujos enunciados serão dados a seguir. Crie um arquivo de testes 
TesteGrafo.hs que importe o módulo Grafo.hs para testar cada uma dessas funções em pelo menos 
três grafos diferentes. As funções são:

éTrivial g, devolve True se o grafo g é trivial ou False, se g não é trivial.

éIsolado g v, devolve True se o vértice v é isolado no grafo g ou False, em caso contrário.

éTerminal g v, devolve True se o vértice v é terminal no grafo gou False, em caso contrário.

éPar g v, devolve True se o vértice v é um vértice par no grafo g ou False, em caso contrário.

éImpar g v, devolve True se o vértice v é um vértice ímpar no grafo g é trivial ou False, em caso contrário.

seqGraus g devolve a sequência de graus de g

grauMax g, devolve o grau máximo de g

grauMin g, devolve o grau mínimo de g

éRegular g, devolve True se o grafo g é regular ou False, em caso contrário.

éKRegular g k, devolve True se o grafo g é k-regular ou False, em caso contrário.

éVazio g, devolve True se o grafo g é vazio ou False, em caso contrário.

éNulo g, devolve True se o grafo g é nulo ou False, em caso contrário.

éCompleto g, devolve True se o grafo g é completo ou False, em caso contrário.

éKn g n, devolve True se o grafo g é completo com n vértices ou False, em caso contrário.

grafoCompleto n, devolve um grafo completo com n vértices (K_n)

grafoComplemento g, devolve um grafo que é o complemento de um grafo g

éSubgrafo h g, devolve True se o grafo h é um subgrafo do grafo g ou False, em caso contrário.

éSubgrafoPróprio h g, devolve True se o grafo h é um subgrafo próprio do grafo h ou False, em caso contrário.

éSubgrafoInduzidoVértices h g vh, devolve True se o grafo h é um subgrafo induzido por um lista de vértices vh do grafo g ou False, em caso contrário.

éSubgrafoInduzidoArestas h g ah, devolve True se o grafo h é um subgrafo induzido por um lista de arestas ah do grafo g ou False, em caso contrário.

éClique h g, devolve True se o grafo h é um clique do grafo g ou False, em caso contrário.

éCjIndependenteVértices h g, devolve True se o grafo h é um conjunto independente de vértices do grafo g ou False, em caso contrário.

união g h, devolve um grafo que é a união dos grafos g e h.

interseção g h, devolve um grafo que é a interseção dos grafos g e h.

soma g h, devolve um grafo que é a soma dos grafos g e h.

removeVértices g vs, devolve um grafo com todos os vértices (e arestas incidentes) na lista vs removidos de g. Dica: para esse exercício e para o próximo já existe uma função nos arquivos dados que remove um único vértice ou uma única aresta. Use-as.

removeArestas g as, devolve um grafo com todos as arestas na lista vs removidas de g
